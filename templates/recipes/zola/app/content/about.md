## What is Zola?

Zola is a fast static site generator written in Rust. It comes as a single binary with no dependencies, making it incredibly easy to install and use.

### Key Features

- **Fast**: Zola generates your entire site in milliseconds
- **Single Binary**: No npm, Ruby gems, or Python packages to manage
- **Sass Built-in**: Write Sass/SCSS directly without additional configuration
- **Shortcodes**: Extend Markdown with custom shortcodes
- **Live Reload**: Development server with instant updates
- **Syntax Highlighting**: Over 100 themes for code highlighting
